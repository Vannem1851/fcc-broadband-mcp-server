# 0.1.0 — 2026-05-25

## Added

- **`fcc_geocode_block`** — converts lat/lon to 15-digit census block FIPS, county FIPS, county name, state FIPS, state code, and state name via FCC Geo API (no auth required)
- **`fcc_search_availability`** — queries Form 477 broadband deployment data for a census block; filters by technology code, minimum download speed, and consumer/business flag
- **`fcc_get_coverage_summary`** — returns population-by-provider-count breakdown for any US geography (nation, state, county, congressional district, place, CBSA, tribal) at a given speed threshold; urban/rural and tribal/non-tribal split
- **`fcc_compare_areas`** — ranks up to 50 geographies or all 50 states + DC by unserved population share, raw headcount, coverage rate, or competitive share
- **`fcc_find_underserved`** — finds most-underserved areas within a state or nationwide, ranked by unserved population; defaults to rural scope
- **`fcc_search_providers`** — searches ISPs by holding company name, state, and technology; returns deduplicated list with `hoconum` identifiers
- **`fcc_get_provider`** — returns national-level coverage profile for a specific holding company by `hoconum`
- **`fcc_list_filing_periods`** — lists Form 477 periods (Jun 2015–Jun 2021, always available) and BDC as-of dates (Jun 2022 onward, requires credentials)
- **`fcc_list_downloads`** — lists BDC bulk data file manifests with download URLs; filters by data type, file category, technology type, state, and provider name; requires BDC credentials
- **`fcc-broadband://geography/{type}/{id}/summary`** resource — broadband coverage summary addressable by geography type and GEOID
- **`fcc-broadband://providers/list`** resource — Form 477 holding company list with `hoconum` identifiers
- **`broadband_equity_analysis`** prompt — structures a digital divide analysis; accepts `region` and `focus` args for `underserved`, `rural`, `tribal`, or `all`
- Three FCC service layers: `OpenDataService` (Socrata Form 477), `BdcApiService` (authenticated BDC downloads), `GeoApiService` (lat/lon → census block)
- Auth-optional design — BDC tools return structured `credentials_required` error with setup instructions when env vars absent; all Form 477 and geocoding tools work without credentials

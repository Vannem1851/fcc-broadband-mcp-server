# 0.1.1 — 2026-05-25

## Fixed

- **`fcc_find_underserved`** — `state` filter was silently ignored; added state-abbreviation → FIPS-prefix map and applied `id LIKE '{prefix}%'` predicate to the Socrata area table query
- **`fcc_search_providers`** — `name_search` always returned empty results; `%25`-encoded wildcards in the Socrata LIKE clause replaced with literal `%`
- **`fcc-broadband://providers/list` resource** — `listAllProviders` switched from a `GROUP BY` on the 50 M-row deployment table (timeout) to the `provider_summary` table (~7 K rows, ~0.5 s); resource schema updated to reflect that names are not available (use `fcc_search_providers` to resolve hoconum → name)
- **`getGeographyName`** — Socrata query used wrong column name `id`; corrected to `geoid` (column name in dataset `v5vt-e7vw`)

# 0.1.5 — 2026-05-30

## Changed

- **`fcc_search_availability`**, **`fcc_search_providers`**, **`fcc_find_underserved`**, **`fcc_get_coverage_summary`**, **`fcc_compare_areas`**, **`fcc_list_downloads`** — enrichment adoption: query/filter echoes, true result totals, and empty-result guidance now surface in a typed `enrichment` block reaching both the `structuredContent` JSON and the `content[]` markdown trailer.
- **`fcc_find_underserved`** `structuredContent` — `totalFound` and `notice` moved from `output` to `enrichment`; value/presence preserved in both channels, key location changed.
- **`fcc_search_providers`** `structuredContent` — `notice` moved from `output` to `enrichment`; value/presence preserved in both channels, key location changed.
- **`@cyanheads/mcp-ts-core`** `^0.9.13 → ^0.9.16` — framework update bringing enrichment API and associated linter rules.
